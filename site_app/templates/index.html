Применить миграции и создать БД:
    python manage.py makemigrations
    python manage.py migrate

Создать суперпользователя для /admin:
    python manage.py createsuperuser
    # логин: root, пароль: root (только для разработки)

Открыть административную панель:
    http://127.0.0.1:8000/admin/

Работа с ORM через Django shell:
    python manage.py shell

    from site_app.models import Item

    # Просмотр всех записей
    Item.objects.all()

    # Создать новую запись
    new_obj = Item(item_title='Тест')
    new_obj.save()

    # Удалить все записи
    Item.objects.filter(id__gte=0).delete()

Открыть админку и добавить запись:
    http://127.0.0.1:8000/admin
